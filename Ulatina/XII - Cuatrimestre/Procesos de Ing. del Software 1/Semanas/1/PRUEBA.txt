Sistemas de información
es todo lo referente al software y al hardware de un sistema, ya sean: datos, procesos
diagramas, documentacion, funcionalidades, requeriemientos

Administrador Proyecto
es la persona responsable de iniciar todos los procesos para el desarrollo de un proyecto, 
ademas de ser el responsable durante el desarrollo

Analista de sistemas
Es un profecional que revisa y estudia las necesidades y requeriemientos del software o sistema
que quiere una empresa

UML
son una serie de diagramas para diseñar un sistema 

Caso de uso
son unas series de pasos que debe realizar el usuario mientras interacctua con el sistema
para completar una accion especifica 

Requerimiento: 
es una funcionalidad que un software se pueden contar como funcional o no funcional

SGBD
son sistemas que gestionan, crean, modifican y mantienen bases de datos
